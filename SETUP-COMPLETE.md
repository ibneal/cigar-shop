## ğŸ¯ Next Steps (In Order)

### 1. Install Node.js (5 minutes)
**If you haven't already:**
- Download from: https://nodejs.org/
- Choose LTS version
- Install with default settings

### 2. Install Dependencies (3 minutes)
```bash
cd cigar-shop
npm install
```

### 3. Add Your Images (15 minutes)
Place images in `public/images/`:
- `logo.png` - Your logo
- `hero-home.jpg` - Home page banner
- `hero-products.jpg` - Products banner
- `hero-about.jpg` - About banner
- `hero-contact.jpg` - Contact banner
- `hero-location.jpg` - Location banner
- `hero-deals.jpg` - Deals banner
- `hero-shop.jpg` - Shop banner
- `categories/cigars.jpg` - Cigar category image
- `categories/vape.jpg` - Vape category image
- `categories/hookah.jpg` - Hookah category image
- `brands/[brand-name].png` - Brand logos

**Don't have images?** Use placeholders for now.

### 4. Update Your Content (10 minutes)
Edit `data/siteConfig.json`:
- Update business hours
- Verify phone number and address
- Check email address
- Update social media links
- Review featured brands
- Verify product descriptions

### 5. Test Locally (5 minutes)
```bash
npm run dev
```
Open: http://localhost:3000

Check:
- All pages load
- Images display
- Navigation works
- Mobile view looks good

### 6. Set Up Services (15 minutes)

**Google Maps API:**
1. Get key: https://console.cloud.google.com
2. Create `.env.local` file
3. Add: `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_key`

**Email Service (for contact form):**
1. Sign up at https://resend.com (free tier)
2. Get API key
3. Add to `.env.local`: `RESEND_API_KEY=your_key`

### 7. Deploy (10 minutes)

**Easiest: Vercel (Free)**
1. Push code to GitHub
2. Sign in to https://vercel.com
3. Import project
4. Add environment variables
5. Deploy!
