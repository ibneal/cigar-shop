# Image Assets Guide

This document lists all the images needed for the website. Place all images in the `public/images/` directory.

## Required Images

### Logo
- `logo.png` - Company logo (transparent PNG, recommended size: 200x200px)

### Hero Images (Full-width banners)
All hero images should be high-quality JPG/WebP files, recommended size: 1920x600px minimum

- `hero-home.jpg` - Home page hero banner
- `hero-products.jpg` - Products page hero (cigar/smoke themed)
- `hero-about.jpg` - About Us page hero
- `hero-contact.jpg` - Contact page hero
- `hero-location.jpg` - Location page hero
- `hero-deals.jpg` - Deals page hero
- `hero-shop.jpg` - Shop page hero

### Store Photos
- `store-front.jpg` - Photo of the store (recommended: 800x600px)
- `cigar-history.jpg` - Historical/educational cigar image
- `tobacco-cultivation.jpg` - Tobacco farming/cultivation image
- `cigar-smoking.jpg` - Cigar being smoked/enjoyed

### Product Category Images
Located in `public/images/categories/`

- `cigars.jpg` - Premium cigars display (800x800px)
- `vape.jpg` - Vaporizer/vape clouds (800x800px)
- `hookah.jpg` - Hookah setup (800x800px)

### Brand Logos
Located in `public/images/brands/`

All brand logos should be transparent PNG files, recommended size: 200x100px

- `cohiba.png`
- `blackstone.png`
- `backwoods.png`
- `djarum.png`
- `raw.png`
- `smok.png`
- `zippo.png`

Add additional brand logos as needed.

## Image Optimization Tips

1. **Format**: Use WebP for photos where possible (smaller file size)
2. **Compression**: Compress images before uploading (use tools like TinyPNG, Squoosh)
3. **Size**: Don't upload images larger than necessary
4. **Naming**: Use lowercase, kebab-case for file names (e.g., `store-front.jpg`)

## Placeholder Images

If you don't have images yet, you can use placeholder services:
- https://placehold.co/ (e.g., https://placehold.co/1920x600)
- https://unsplash.com/ (free stock photos)
- https://pexels.com/ (free stock photos)
